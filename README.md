    - [Pandas для вирішення вищезгаданих завданнь](test_python.ipynb) - Підвантаження через Pandas *.csv які використовував для створення БД в SQL Server.

# Висновок

Хоча тестові дані далекі від ідеалу з точки зору семантики і в реальному проекті такі зв'язки та моделі даних об'єднувати не варто, особливо при агрегації фінансових показників, намагався подати результати максимально інформативно.
Результати тестового завдання виконані без уточнення деталей, тому описані рішення є суб'єктивними та демонстраційними.
